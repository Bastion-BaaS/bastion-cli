### Bastion CLI deploys the infrastructure that your Bastion Instances run on

# Usage

```bash
$ npm install -g bastion-cli
$ bastion <COMMAND>
running command...
$ bastion (-v|--version|version)

bastion-cli/0.1.0 darwin-arm64 node-v16.13.0

$ bastion --help [COMMAND]
USAGE
  $ bastion COMMAND
...
```

# Commands

- `bastion deploy`
- `bastion destroy`
- `bastion help [COMMAND]`
- `bastion config`
